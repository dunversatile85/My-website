## Style Guidelines:

- Primary color: Deep sky blue (#00BFFF) for an immersive feel.
- Background color: Light cyan (#E0FFFF), offering a light but themed frame to the experience.
- Accent color: Electric blue (#7DF9FF) for interactive elements such as buttons and icons.
- Font: 'Inter' sans-serif for a clean, readable UI, suitable for all text elements. Note: currently only Google Fonts are supported.
- Use bright, modern icons that correspond with rewarded actions.
- Layout should provide prominent visibility to 'Earn Rewards', and 'Watch Ad' buttons.
- Use subtle animations to highlight reward availability and notify users upon task completion.
